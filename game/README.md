# Game

게임과 관련된 다양한 정보, 도구, 프로젝트를 정리하는 토이 프로젝트 폴더입니다.

## 목적

- 게임 관련 데이터 수집 및 분석 도구 제작
- 커스텀 게임 엔진 개발 (C++ / DirectX)
- 향후 AI 기반 게임 관련 기능 실험

## 구조

```
game/
├── KojeomEngine/   # 커스텀 게임 엔진 (C++ / DirectX)
└── diablo2/        # Diablo 2 Resurrected 관련 데이터 및 도구
```

## 프로젝트

### KojeomEngine
C++과 DirectX 기반으로 직접 제작 중인 커스텀 게임 엔진.
기본적인 렌더링 파이프라인, 카메라, 메시, 셰이더 등의 시스템을 구현하고 있으며,
예제 프로젝트를 통해 기능을 검증합니다.

### diablo2
Diablo 2 Resurrected의 룬워드(Runeword) 정보를 정리하고 엑셀로 출력하는 도구.

| 파일 | 설명 |
|------|------|
| `create_runeword_excel.py` | 룬워드 데이터를 수집해 엑셀 파일로 생성하는 스크립트 |
| `D2R_룬워드_목록.xlsx` | 생성된 룬워드 목록 엑셀 |

## 향후 계획

- AI를 활용한 게임 내 데이터 분석 및 자동화 실험
- 게임 엔진에 AI 요소 접목 (NPC 행동, 절차적 생성 등)
