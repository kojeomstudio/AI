# 모딩에 필요한 기술 (Required Skills)

D2R 모드를 개발하기 위해 익히면 좋은 기술들입니다.

## 1. 데이터 분석 및 편집 (Excel/CSV)
*   **탭 구분 파일(TSV) 이해:** `.txt` 파일은 엑셀 형식이지만 실제로는 탭으로 구분된 텍스트입니다. 형식을 깨뜨리지 않고 편집하는 능력이 중요합니다.
*   **상호 참조 이해:** 예를 들어 `treasureclassex.txt`에서 아이템 이름을 참조할 때, 해당 아이템이 `weapons.txt`나 `armor.txt`에 올바르게 정의되어 있어야 합니다.

## 2. JSON 문법
*   D2R의 UI와 새로운 엔진 설정은 JSON을 사용합니다. 올바른 문법(중괄호, 쉼표 등)을 지키며 편집하는 능력이 필요합니다.

## 3. 그래픽 편집 능력
*   아이콘을 새로 만들거나 UI를 변경하고 싶다면 Photoshop이나 GIMP 같은 도구를 사용하여 배경이 투명한 PNG 등을 제작하고 스프라이트로 변환하는 과정이 필요합니다.

## 4. 논리적 사고 (Logic)
*   드랍률 계산, 스킬 데미지 공식, 룬워드 조합 조건 등을 설정할 때 게임 로직을 이해하고 수식을 적용하는 능력이 필요합니다.

## 5. 커뮤니티 활용 능력
*   D2R 모딩은 20년 이상의 역사를 가진 원작 디아블로 2의 지식에 기반합니다. **The Phrozen Keep** 같은 커뮤니티에서 기존 가이드를 찾아보고 적용하는 능력이 매우 중요합니다.
