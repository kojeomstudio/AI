# D2R 모딩 도구 (Tools)

D2R 모딩에 필수적이거나 유용한 도구들입니다.

## 1. 데이터 추출 및 보기
*   **CascView (Ladik's):** 게임의 CASC 스토리지에서 파일을 추출하는 데 필수적인 도구입니다.
*   **MPQEditor:** 구버전 디아블로 2와의 호환 작업이나 특정 아카이브 확인 시 사용됩니다.

## 2. 데이터 편집 (Excel/Text)
D2R의 핵심 로직은 `.txt` 파일(탭으로 구분된 텍스트 파일)에 저장되어 있습니다.
*   **AFJ Sheet Editor:** 디아블로 2 모딩에 최적화된 시트 에디터입니다.
*   **D2 Excel:** 고전적인 엑셀 파일 편집 도구입니다.
*   **Microsoft Excel / Google Sheets:** 파일을 편집할 수 있지만, 저장 시 탭 구분 형식을 엄격히 유지해야 합니다.
*   **Visual Studio Code / Notepad++:** `.json` 파일 및 일반 텍스트 편집에 사용됩니다.

## 3. 그래픽 및 UI 편집
*   **SpriteEdit:** D2R의 `.sprite` 파일을 편집하여 아이콘이나 UI 요소를 수정합니다.
*   **Noesis:** 3D 모델 및 `.texture` 파일을 확인하고 변환하는 데 사용됩니다.
*   **Adobe Photoshop / GIMP:** 그래픽 자산을 편집하는 데 사용됩니다.

## 4. 모드 관리 및 기타
*   **D2RMM (D2R Mod Manager):** 여러 모드를 쉽게 설치하고 관리하며, 충돌을 방지하는 도구입니다.
*   **D2Runewizard (Hero Editor):** 캐릭터 세이브 파일(`.d2s`)을 편집하여 테스트 환경을 구축하는 데 유용합니다.
