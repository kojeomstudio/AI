# D2R 모딩 시작하기 (Getting Started)

디아블로 2 레저렉션(D2R) 모딩은 게임의 데이터를 추출하고, 수정된 파일을 특정 폴더 구조에 배치한 뒤, 게임 실행 옵션을 통해 해당 데이터를 불러오는 방식으로 이루어집니다.

## 1. 기본 개념
D2R은 **CASC(Content Addressable Storage Container)** 형식을 사용하여 게임 데이터를 저장합니다. 모딩을 위해서는 이 컨테이너에서 데이터를 추출해야 합니다.

## 2. 데이터 추출 (Extraction)
*   **CascView** 도구를 사용하여 게임 설치 폴더의 데이터를 엽니다.
*   `data/global/excel` (아이템, 스킬 등 로직 데이터)
*   `data/local/lng` (언어/텍스트 데이터)
*   `data/hd` (현대적인 그래픽 및 UI 데이터)
위 폴더들을 추출하여 작업 디렉토리에 저장합니다.

## 3. 모드 파일 구조 (Mod Structure)
D2R은 두 가지 방식으로 모드를 불러올 수 있습니다.

### 방법 A: Direct 모드 (비권장)
게임 설치 폴더의 `Data` 폴더에 직접 파일을 넣는 방식입니다.
*   경로: `Diablo II Resurrected\Data\global\excel\...`
*   실행 옵션: `-direct -txt`

### 방법 B: Mods 폴더 사용 (권장)
별도의 모드 폴더를 생성하여 관리하는 방식입니다.
*   경로: `Diablo II Resurrected\mods\{모드이름}\{모드이름}.mpq\data\...`
    *   참고: 실제로는 `.mpq` 확장자를 가진 폴더를 생성합니다.
*   실행 옵션: `-mod {모드이름} -txt`

## 4. 기본적인 작업 흐름
1.  **데이터 추출:** 필요한 `.txt` 또는 `.json` 파일을 추출합니다.
2.  **데이터 수정:** 엑셀 또는 텍스트 에디터를 사용하여 값을 수정합니다.
3.  **파일 배치:** 위에서 설명한 모드 폴더 구조에 맞춰 파일을 배치합니다.
4.  **게임 실행:** 배틀넷 앱의 '설정' -> '게임 설정' -> '추가 명령줄 인수'에 실행 옵션을 추가하고 게임을 실행합니다.
